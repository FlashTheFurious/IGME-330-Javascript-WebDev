Noteworthy

  1. Audio analyser node was used to get Frequency data about the song
  2. We currently visualize 512 samples on the screen, can be increased by changing the bufferLength variable
  3. App has over 15 control options for the user to customize their experience
  4. Tweaked all shape visualizers to better represent the sound than the code we had in HW2
  5. Bulma is used to make the site mobile friendly
  6. 3 Web Components are used to follow DRY code principles, - Navigation, Header and Footer
  7. Json regarding the color values of the presets is loaded by use of fetch()
